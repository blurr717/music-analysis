```python
def fmt_mmss(sec, _=None):
    """将秒数转为 m:ss 格式"""
    m = int(sec // 60)
    s = int(sec % 60)
    return f"{m}:{s:02d}"

def set_time_axis(ax, duration_sec):
    """将 x 轴设为 mm:ss 格式，每10秒一个刻度"""
    tick_secs = np.arange(0, duration_sec + 1, 10)
    ax.set_xticks(tick_secs)
    ax.set_xticklabels([fmt_mmss(t) for t in tick_secs], fontsize=8)
    ax.set_xlim(0, duration_sec)
```

## 分析维度

| 维度 | 特征 | 解读 |
|------|------|------|
| **能量（Energy）** | RMS、节奏(BPM)、频谱质心、过零率 | 感知强度/活跃度 |
| **效价（Valence）** | MFCC、色度（大/小调）、频谱对比度 | 情绪正负向 |
| **唤醒度（Arousal）** | RMS方差、频谱通量 | 兴奋/紧张程度 |

## 情绪象限映射

| 唤醒度 \ 效价 | 高效价 (>0.6) | 低效价 (<0.4) |
|--------------|--------------|--------------|
| **高唤醒 (>0.6)** | 兴奋/欢快 | 紧张/愤怒 |
| **低唤醒 (<0.4)** | 轻松/愉悦 | 悲伤/忧郁 |

## 分析参数
- 采样率：22050 Hz

- 时间曲线分段：每 2 秒一段
